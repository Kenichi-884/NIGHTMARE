
【3Dモデル】日本の道路パック / Japanese Street Pack 利用規約

-------------------
VRChatのワールドやゲーム開発等に利用可能な3Dモデルです。
Unity2018 / VRCSDK2&3両対応
Substance Painterで制作したリアルなテクスチャ入り。

-------------------
【 内容物 】
UnityPackage形式
モデルfbxファイル及びPrefab　全49種くらい
	・バンプゲート　通常版・汚れ版　2種類
	・バス停
	・長椅子
	・ガードパイプ
	・ガードレール
	・駐車ポール　通常版・汚れ版　2種類
	・路面標識　32種
		中央線　3種類

		駐車場の線　2種類(単体・セット)

サンプルシーン　1個

今後作ったものも順次追加していく予定です。

-------------------
【 使用方法 】
1. UnityPackageをUnityにインポートして下さい。
2. プロジェクトフォルダ内のTsubokuLab/Models/JapaneseStreetPack/以下にある各Prefabをシーンに配置して下さい。
3. シーンを再生して動作を確認して下さい。
4. VRChatのワールドとしてアップロードしたり、ゲームの背景等にご自由にお使い下さい。

■サンプルシーンについて
サンプルシーンを同梱しています。「JapaneseStreetPack_Example.unity」
Bloom等の演出の為にカメラにPostProcessing v2.1.1 を利用しています。
最適な状態で見るにはUnityのPackageManagerからPostProcessing v2.1.1をプロジェクトに追加して下さい。

※ 反射を綺麗に表現するには、ライティング及びReflection Probeの設定を行って下さい。
※ テクスチャ解像度が高めなので、容量を気にする場合は必要に応じてInspectorより解像度変更・画像圧縮等を行って下さい。
